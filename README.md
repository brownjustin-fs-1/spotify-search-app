# Spotify Search App

## Project Overview

This project is a full-stack Spotify Search application that allows users to search for artists, albums, and tracks using Spotify's REST API. The frontend will communicate with a Node.js/Express backend that securely handles authentication using environment variables.

### Planned Features

- Spotify user authentication
- Search for artists
- Search for albums
- Search for tracks
- Display search results
- Responsive user interface

---

## Prerequisites

Before running this project, install:

- Node.js 20+
- npm
- Git
- VS Code

---

## Getting Started

Clone the repository:

```bash
git clone <repository-url>
```

Navigate into the backend folder:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
